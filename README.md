# BuildGTA — Toronto City Builder

Playable 3D Toronto city-builder game (static HTML).

## Play locally
Open `index.html` in a modern browser, or serve the folder:

```
npx serve .
```

## Controls
- Click tools / zones to build
- Drag to pan, scroll to zoom (camera controls in-game)

## Deploy
This repo is set up for Netlify: publish the repo root (`index.html` is the site entry).

Live: connect this GitHub repo to Netlify, or run `npx netlify-cli deploy --prod --dir .` from this folder.
